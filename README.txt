How to run:
1. Clone solution in VisualStudio (https://github.com/rwasik/CashMachine.git) or download it as ZIP 
2. Open CashMachine solution
3. Build solution
4. Set CashMachineApi as startup project
5. Run CashMachineApi using Ctrl+F5
6. Set CashMachineApp as startup project
7. Run CashMachineApp using Ctrl+F5
